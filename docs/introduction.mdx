## Why use NOMOS?

NOMOS helps developers build sophisticated AI agents through structured workflows and configurable components, making complex agent development accessible to teams of all skill levels.

Use NOMOS for:
- **Multi-step agent workflows** - Create complex, stateful interactions with step-by-step flow definitions
- **Specialized tools** - Each step can have its own tools and intelligent routing between conversation states
- **Multiple LLM support** - Works with OpenAI, Anthropic, Google, Mistral, Ollama, and HuggingFace
- **Flexible deployment** - From simple scripts to production servers with Redis/PostgreSQL

## Quick Installation

<CodeGroup>

```bash Basic Installation
pip install nomos
```

```bash With CLI Support
pip install nomos[cli]
```

```bash With OpenAI Support
pip install nomos[openai]
```

```bash Full Installation
pip install nomos[cli,openai,serve,traces]
```

</CodeGroup>

## TypeScript/JavaScript SDK

For client-side applications and web development:

```bash
npm install nomos-sdk
```
